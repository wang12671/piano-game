# 生活效率工具箱

纯前端静态多工具网站，无后端依赖，可直接部署 GitHub Pages 实现在线访问。

## 在线访问

**GitHub Pages URL**: `https://[username].github.io/[repo]`

> 部署方法：将 `index.html` 上传到 GitHub 仓库 → Settings → Pages → Source 选择 main 分支

## 功能清单

### 工具 1：随机抽签抽奖
- 批量导入抽奖名单（换行分隔）
- 设置抽取人数
- 开关控制是否允许重复抽取
- 滚动动画抽奖效果
- 中奖结果高亮展示
- 历史记录保存
- 一键复制结果
- 导出抽奖历史为 CSV

### 工具 2：待办清单
- 添加/删除待办事项
- 标记完成状态
- 待完成/已完成数量统计
- 批量导入/导出 CSV
- 数据本地持久化

### 工具 3：收支记账
- 记录收入/支出
- 金额、分类、日期、备注
- 月度收支柱状图
- 账单列表管理
- 导出账单为 CSV

### 关于我
- 个人简历页面
- 技能栈展示
- 项目经历
- 联系方式

### 全局功能
- **数据持久化**：所有数据自动存入 localStorage，刷新页面不丢失
- **全局导入导出**：一键备份/恢复所有工具数据为 JSON 文件
- **使用统计面板**：展示各工具累计使用次数
- **数据可视化**：待办完成率饼图、月度收支柱状图
- **深色/浅色主题**：一键切换全站配色
- **快捷入口**：自动记录最近使用工具

## 使用教程

### 抽奖工具
1. 在名单框输入参与人员（每行一个）
2. 设置抽取人数
3. 点击"开始抽奖"
4. 查看结果，复制或查看历史

### 待办清单
1. 输入待办内容，按 Enter 或点击添加
2. 点击圆形勾选标记完成
3. 悬停显示删除按钮

### 记账工具
1. 选择收入/支出类型
2. 输入金额和分类
3. 点击添加记录
4. 查看统计和图表

### 数据备份
1. 点击"导出全部数据"下载 JSON 备份
2. 需要恢复时点击"导入数据"选择备份文件

## 技术栈
